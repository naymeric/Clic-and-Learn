CSS c'est quoi ?

CSS, ou Cascading Style Sheets, est un langage utilisé pour styliser et mettre en forme les pages web. CSS permet de contrôler l'apparence des éléments HTML, comme les couleurs, les polices, les espacements, les marges, et bien plus encore.

Squelette d'une page CSS

Dans un code CSS vous allez avoir 3 éléments importants et 1 notion à retenir. Ces éléments sont :

  • le sélecteur,
  • la propriété,
  • la valeur

La propriété couplée à une valeur induit la notion de règle (ou déclaration) qui détermine la modification apportée à un élément HTML déterminé par un sélecteur.

Sélecteurs

CSS utilise des sélecteurs pour cibler les éléments HTML que tu veux styliser. Par exemple, un sélecteur peut cibler toutes les balises "p" (paragraphes) ou une classe spécifique comme .ma-classe.

Zoom sur les sélecteurs en CSS

Type de sélecteur Fonction du sélecteur Exemple
Sélecteurs de type Sélectionne tous les éléments d'un type donné
Sélecteurs de classe Cible les éléments avec une classe spécifique précédée d'un point (.)
Sélecteurs d'identitifant Cible un élément avec un identifiant spécifique précédé d’un dièse (#)
Sélecteurs descendant Sélectionne les éléments qui sont des descendants d'un autre élément
Sélecteurs d'attribut Cible les éléments qui possèdent un certain attribut
Sélecteurs de pseudo-classe Cible les éléments dans un état particulier

Type de propriété Fonction de la propriété Exemple

Display Contrôle le type d'affichage d'un élément
Text-align Définit l'alignement du texte
