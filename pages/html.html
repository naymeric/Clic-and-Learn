Le HTML c'est quoi ?

HTML, ou HyperText Markup Language, est le langage standard utilisé pour créer des pages web. Il permet de structurer le contenu en utilisant des balises, comme pour les titres ou pour les paragraphes. En gros, HTML définit la structure de la page, tandis que d'autres langages comme CSS et JavaScript s'occupent de la présentation et de l'interactivité. C'est la base de tout site internet !

Les balises de structure d'une page HTML

  • !DOCTYPE : Indique que le document est en HTML5.
  • HTML : L'élément racine qui encapsule tout le document.
  • HEAD : contient des informations sur la page (métadonnées, liens vers des styles et scripts).
  • BODY : Contenu visible de la page web.
  • HEADER : Contient le titre et la navigation.
  • MAIN : Partie principale où se trouve le contenu.
  • MAIN : Partie principale où se trouve le contenu.
  • SECTION : Sections distinctes pour organiser le contenu.
  • FOOTER : Contient des informations supplémentaires, comme des droits d'auteur.
